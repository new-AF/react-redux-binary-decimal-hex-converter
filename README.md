# Unsigned Binary - Hexadecimal - Decimal number converter

App to convert

- binary to decimal.
- binary to hexadecimal.

and

- decimal to binary.
- decimal to hexadecimal.

and

- hexadecimal to binary.
- hexadecimal to decimal.
